Вывод данных - Уравнение связи

После выполнения сортировки и сбора данных программа представляет следующие результаты:

1. Таблица с исходными данными

Для каждой итерации с размером от 9000 до 50000 элементов будет заполнена таблица, которая содержит следующие данные:

  • Номер: Индекс текущего номера наблюдения
  • Время (y): Время, затраченное на сортировку массива
  • Размер массива (x): Размер массива, сортируемого в данной итерации.
  • x * x: Квадрат размера массива.
  • x * y: Произведение размера массива и времени сортировки.
  • y * y: Квадрат времени сортировки.

2. Построение графика

Будет построен график, показывающий зависимость времени сортировки от размера массива.

3. Расчёт коэффициентов уравнения связи

На основе собранных данных программа рассчитывает два коэффициента уравнения связи:

  • a0: свободный коэффициент
  • a1: выборочный коэффициент

4. Уравнение связи

Программа вычисляет уравнение связи между временем сортировки и размером массива.

5. Анализ уравнения связи

После построения уравнения связи, программа выводит анализ уравнения связи, который включает в себя расчёт коэффициентов парной корреляции, детерминации, эластичности, и бета-коэффициент

  • Коэффициент корреляции: Показывает, насколько сильна линейная зависимость между временем сортировки и размером массива. Чем ближе значение к 1, тем сильнее связь.
  • Коэффициент детерминации: Показывает значимость входной переменной х. Это доля вариации выходной переменной, объяснённой экзогенной переменной x.
  • Коэффициент эластичности: Оценивает, на сколько процентов изменится время сортировки при изменении размера массива на 1%
  • Бета-коэффициент: Измеряет изменение времени сортировки при изменении размера массива на одно стандартное отклонение.

6. Линия регрессии

После выполнения расчётов будет построена линия регрессии, которая отображает линейную зависимоить между временем сортировки и размером массива на графике
