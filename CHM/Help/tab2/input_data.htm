Ввод данных - Уравнение связи

Для расчёта уравнения связи между временным и ёмкостным параметрами алгоритма необходимо ввести следующие данные:

1. Объём выборки

Укажите размер выборки, для которой будет рассчитано уравнение связи. Это число определяет количество элементов, используемых для оценки зависимости между временным и ёмкостными параметром.

Примечание: Коэффициенты для обратной функции Брэдфорда заданы по умолчанию:

  • A = 0,5
  • B = 1
  • C= 1
