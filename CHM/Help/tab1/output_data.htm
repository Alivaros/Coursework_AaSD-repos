Вывод данных - Сортировка методом Timsort

После выполнения сортировки методом Timsort, программа выводит следующие данные:

1. Таблица с данными

Будут заполнены таблицы, которые содержат:

  • Исходный массив: массив данных до выполнения сортировки.
  • Упорядоченный массив: массив, отсортированный с помощью алгоритма Timsort.

2. Выходные данные

После сортировки будут выведены следующие статистические данные:

  • Время сортировки: время, затраченное на выполнение сортировки. Это значение зависит от количества элементов в массиве
  • Количество сравнений: число сравнений между элементами массива, выполенных в процессе сортировки.
  • Количество перестановок: число перестановок элементов массива, произведённых в процессе сортировки.

После вывода всех данных, вы сможете увидеть результат работы программы и оценить эффективность алгоритма сортировки для вашего набора данных.
