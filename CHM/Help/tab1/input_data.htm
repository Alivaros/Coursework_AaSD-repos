Ввод данных - Сортировка методом Timsort

Для запуска алгоритма сортировки методом Timsort, необходимо ввести следующие данные:

1. Количество элементов массива (n)

Укажите целое число, которое будет определять размер массива, подлежащего сортировке. Количество элементов может быть любым, но чем больше элементов, тем больше времени потребуется для сортировки.

2. Коэффициенты для обратной функции Брэдфорда (A,B,C)

Эти коэффициенты используются для генерации случайных чисел, которые будут заполнять массив. При вводе данных необходимо учитывать следующие ограничения:

  • Коэффициент A: должен быть меньше коэффициента B (A < B).
  • Коэффициент B: должен быть больше коэффициента A (B < A).
  • Коэффициент C: должен быть больше нуля (C > 0).

После того как все данные введены, программа будет использовать эти параметры для генерации массива и сортировки его с помощью метода Timsort. Если введены некорректные данные, программа не позволит продолжить процесс сортировки. При правильном вводе значений, кнопка "Сортировка" будет активной.
